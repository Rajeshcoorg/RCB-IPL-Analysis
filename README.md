# RCB IPL 2017 Data Analysis & Strategy Project

## Overview
This project analyzes Royal Challengers Bangalore (RCB) performance using IPL 2017 dataset. The goal is to understand player performance, match conditions, and key factors affecting wins and losses.

## Objectives
- Find top batsmen and bowlers  
- Analyze death over performance  
- Study home ground performance  
- Identify all-rounders and key contributors  

## Dataset
Data is taken from IPL SQL tables such as:
- Matches  
- Ball_by_Ball  
- Player  
- Team  

## Methodology
- Data cleaning and filtering for RCB  
- Used SQL (JOIN, GROUP BY, aggregation functions)  
- Analyzed performance using queries  
- Visualized results using Google Sheets  

## Key Insights
- RCB depends heavily on top-order batsmen  
- Death over bowling needs improvement  
- Inconsistent performance at home ground  
- Few players contribute most of the performance
